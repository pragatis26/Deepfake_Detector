About This Project

AI-Based DeepFake Detection for Digital Media Authentication

Project Overview

DeepFake technology — AI-generated synthetic media that replaces one person's likeness with another's — poses serious threats to digital trust, journalism, and security. This project presents a deep learning pipeline to automatically detect deepfake images and videos using a fine-tuned EfficientNet-B4 convolutional neural network, augmented with Grad-CAM explainability.

Model Architecture

Base ModelEfficientNet-B4 (pretrained on ImageNet)
Input Size224 × 224 × 3 (RGB)
OutputBinary (Real / Fake) via Sigmoid
Face DetectorMTCNN (facenet-pytorch)
ExplainabilityGrad-CAM on last conv block
Training DatasetFaceForensics++ (FF++)
FrameworkPyTorch 2.x

Training Details

  • Phase 1 (Epochs 1–3): Only the classifier head is trained (feature extraction). Base EfficientNet layers are frozen.
  • Phase 2 (Epochs 4+): Full fine-tuning — all layers unfrozen with a 10× smaller learning rate.
  • Augmentations: Random crop, horizontal flip, colour jitter, rotation.
  • Loss: Binary Cross-Entropy (BCELoss)
  • Optimizer: Adam with CosineAnnealing LR scheduler
  • Best model selection: Highest Val ROC-AUC
